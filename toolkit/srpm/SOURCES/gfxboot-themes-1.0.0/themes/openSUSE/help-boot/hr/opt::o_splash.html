Izbori kernela: splash

Pozdravni ekran je slika koja se prikazuje prilikom podizanja sustava.

  • splash=0 -- pozdravni ekran je isključen. To može biti korisno na starim monitorima ili ako se desi neka pogreška.
  • splash=verbose -- aktivira pozdravni ekran, poruke kernela i podizanja sustava se prikazuju.
  • splash=silent -- Aktivira pozdravni ekran, ali bez poruka. Umjesto toga se koristi traka koja prikazuje napredovanje.
