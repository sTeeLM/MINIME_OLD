การใช้ระบบช่วยเหลือ

ระบบช่วยเหลือของตัวจัดการการบูตนั้นผันแปรได้ โดยมันจะให้ข้อมูลเกี่ยวกับรายการเมนูที่เลือกไว้ หรือ หากคุณทำการแก้ไขตัวเลือกการบูต มันจะพยายามหาข้อมูลเกี่ยวกับตัวเลือกที่มีเคอร์เซอร์เลือกอยู่

ปุ่มพิมพ์ควบคุม

  • ปุ่มลูกศรขึ้น: เน้นไปยังการเชื่อมโยงก่อนหน้า
  • ปุ่มลูกศรลง: เน้นไปยังการเชื่อมโยงถัดไป
  • ปุ่มลูกศรซ้าย, ปุ่ม Backspace: ย้อนกลับไปยังหัวข้อก่อนหน้านี้
  • ปุ่มลูกศรขวา, ปุ่ม Enter, ปุ่ม Space: ตามการเชื่อมโยง
  • ปุ่ม Page Up: เลื่อนขึ้นหนึ่งหน้า
  • ปุ่ม Page Down: เลื่อนลงหนึ่งหน้า
  • ปุ่ม Home: ไปยังหน้าเริ่มต้น
  • ปุ่ม End: ไปยังหน้าสุดท้าย
  • ปุ่ม Esc: ออกจากความช่วยเหลือ
