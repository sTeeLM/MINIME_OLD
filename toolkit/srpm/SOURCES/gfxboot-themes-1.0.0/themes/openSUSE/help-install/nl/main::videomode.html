Videomodus selecteren

Druk op @@@fkey_video@@@ om een lijst met videomodi op te halen die door uw grafische kaart worden ondersteund. De hoogste modus die uw monitor kan weergeven, is voorgeselecteerd.

Het is mogelijk dat uw monitor niet automatisch kan worden gedetecteerd. In dat geval kunt u hier de juiste modus handmatig selecteren.

Als u tijdens de installatie problemen ondervindt met de grafische kaart, kan tekstmodus een bruikbare alternatieve methode zijn.
