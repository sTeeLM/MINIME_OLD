Kernelopties: splash

Het splash-scherm is de afbeelding die wordt getoond tijdens het opstarten van het systeem.

  • splash=0 -- Het splash-scherm wordt uitgeschakeld. Dit kan handig zijn voor oude beeldschermen of als er fouten optreden.
  • splash=verbose -- Activeert splash en kernel- en opstartberichten worden nog weergegeven.
  • splash=silent -- Activeert splash, maar zonder berichten. In plaats daarvan wordt een voortgangsbalk getekend.
