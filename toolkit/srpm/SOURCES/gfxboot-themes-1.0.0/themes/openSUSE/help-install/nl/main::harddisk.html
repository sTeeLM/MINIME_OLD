Van vaste schijf opstarten

Selecteer Geïnstalleerd besturingssysteem starten om een systeem te starten dat op uw lokale vaste schijf is geïnstalleerd. Dit systeem moet goed zijn geïnstalleerd, omdat alleen de MBR (Master Boot Record) op de eerste vaste schijf wordt gestart. De apparaat-ID van de eerste vaste schijf wordt geleverd door het BIOS van de computer.

Gebruik dit als u vergeten bent de cd of dvd uit uw station te verwijderen en de computer vanaf de vaste schijf wilt opstarten.
