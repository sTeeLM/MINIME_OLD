Geheugentest

De ingesloten Geheugentest levert goede mogelijkheden tot een stresstest van de hardware van een systeem. Het hoofddoel is om defect RAM-geheugen te ontdekken, maar het test ook vele andere onderdelen van het systeem.

Er is geen garantie dat het geheugen goed is als er geen fouten zijn gevonden, hoewel de meeste fouten in het geheugen wel zullen worden gevonden.
